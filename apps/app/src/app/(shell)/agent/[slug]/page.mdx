## Step 2: Connect using your favorite framework

First, install the required package:

<CodeGroup>
```bash Vercel AI SDK
$ npm install ai
```

```bash Langchain
$ pip install -U langchain-openai portkey-ai
```

</CodeGroup>

Then, create the following code:

<CodeGroup>
```ts Vercel AI SDK
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { generateText } from 'ai';

const provider = createOpenAICompatible({
apiKey: 'YOUR_API_KEY',
baseURL: 'https://cloud.hebo.ai/api',
});

const response = await generateText(
model: provider('YOUR_MODEL_ALIAS'),
{ prompt: 'Tell me a joke about AI.' }
);
console.log(result);

````

```python Langchain
from langchain_openai import ChatOpenAI
import os

portkey_headers = createHeaders(
    api_key="YOUR_API_KEY",
    provider="openai"
)

llm = ChatOpenAI(
    base_url="https://cloud.hebo.ai/api",
    model="YOUR_MODEL_ALIAS"
)

response = llm.invoke("Tell me a joke about AI.")
print(response.content)
````

</CodeGroup>
