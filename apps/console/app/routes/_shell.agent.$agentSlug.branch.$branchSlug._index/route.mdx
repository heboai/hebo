## Step 2: Connect using your favorite framework

First, install the required package:

<CodeGroup>
```bash Vercel AI SDK
npm install ai @ai-sdk/openai-compatible
```

```bash Langchain
pip install -U langchain-openai
```
</CodeGroup>

Then, create the following code:

<CodeGroup>
```ts Vercel AI SDK
import { generateText } from 'ai';
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';

const provider = createOpenAICompatible({
    name: 'hebo',
    apiKey: 'sk_pf***',  // Replace with your real hebo api key
    baseURL: 'https://gateway.hebo.ai/v1',
});

const { text } = await generateText({
    model: provider('my-agent/main/default'),  // Replace with your real model alias
    prompt: 'Tell me a joke about AI.'
});

console.log(text);
```

```python Langchain
from langchain_openai import ChatOpenAI

llm = ChatOpenAI(
    model="my-agent/main/default",  # Replace with your real model alias
    api_key="sk_pf***",  # Replace with your real hebo api key
    base_url="https://gateway.hebo.ai/v1",
)

response = llm.invoke("Tell me a joke about AI.")
print(response.content)
```
</CodeGroup>
